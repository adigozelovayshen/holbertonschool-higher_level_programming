# Python - Abstract Base Classes (ABC)

## Description
This project focuses on understanding and implementing **Abstract Base Classes (ABC)** and **Duck Typing** in Python. It includes:
* Defining abstract methods using the `abc` module.
* Implementing concrete classes (`Circle`, `Rectangle`) that inherit from an abstract class (`Shape`).
* Demonstrating duck typing through a standalone function `shape_info`.

## Files
| File | Description |
| --- | --- |
| task_01_duck_typing.py | Contains Shape, Circle, Rectangle classes and shape_info function. |

## Requirements
