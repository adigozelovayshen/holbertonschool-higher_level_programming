# Python - Abstract Classes and Interfaces

This project explores object-oriented programming concepts in Python including abstract classes, interfaces, duck typing, and subclassing.

## Tasks

### 0. Abstract Animal Class and its Subclasses
- Abstract class `Animal` with abstract method `sound`
- Concrete classes `Dog` and `Cat`

### 1. Shapes, Interfaces, and Duck Typing
- Abstract class `Shape` with abstract methods `area` and `perimeter`
- Concrete classes `Circle` and `Rectangle`
- `shape_info` function using duck typing
